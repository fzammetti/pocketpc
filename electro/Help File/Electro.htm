Introduction
Electro is a comprehensive collection of useful tools for the electronics technician and electrician.

It is organized into two sections, Tools and Reference:

Tools - This menu contains a number of calculators and utilities to help in design and engineering tasks.

Reference - This menu contains various reference tables and collections of information that may be helpful in day-to-day electronics and electrical work.

You can move around the program by selecting either the Tools or Reference menu and then selecting the item you'd like to use. You can freely move around to various screens at any time, there is no need to close each individual screen when you are done. In fact, if you use a particular tool and go to another and later return to the first, it will be exactly as you left it, you won't loose an calculations or anything else as you move from screen to screen.

You can access help for the screen you are currently viewing by clicking the question mark on the toolbar. You can of course always view the complete help file by selecting Help... under the Help menu.

You can exit Electro at any time by clicking Exit on the Tools menu.

If you find any bugs in the program, please send eMail to fzammetti@omnytex.com. You can also send any suggestions for additions to that same address. We are always thinking of future versions and what we can put in to make Electro the best program of it's kind for the PopketPC.

Thank you very much for your purchase and we hope you fine Electro helpful in your work!

555 Timer Design
With this tool you can design astable (one-shot) and monostable (multivibrator) 555 timer circuits.

Begin by selecting what mode you want to calculate for, Monostable or Astable, by clicking one of the tabs.

For Monostable mode, enter the following::

Ra (Ohms)
C1 (Farads)

Click the Calculate button and the time in seconds will be calculated for you.

For Astable mode, enter the following:

R1 (Ohms)
R2 (Ohms)
C1 (Farads)

Again click the Calculate buttons and the waveform's High Time (Th) and Low Time (Tl) will be calculated, as well as the frequency.

Click Clear at any time to clear any current values and start over.


DC Ohm's Law
This tool is used to perform various calculations based on Ohm's Law for DC circuits.

To use this calculator, simply enter any two known values from the following:

Current (I, Amperes)
Voltage (E, Volts)
Resistance (R, Ohms)
Power(P, Watts)
Then click the Calculate button and the other two values will be calculated and shown in their respective boxes. Click Clear at any time to clear any current values and start over.


Delta-Wye Transforms
This tool is used to calculate the resistor values when performing a transformation from a Delta configuration to a Wye configuration or vice-versa.

To use this calculator, first make sure the arrow between the Delta and Wye diagrams is pointing the way you want the transformation to go (initially it is set for Delta-to-Wye). Simply click the arrow button to reverse the direction. The three items appropriate for the source configuration will be enabled while the other three will be disabled. Enter the values for the source (Za, Zb, Zc or Z1, Z3, Z3) and click Calculate. The other three values will be calculated and shown in their respective boxes. Click Clear at any time to clear any current values and start over.


Energy Stored
This tool is used to calculate the energy stored in a capacitor or inductos.

To use this calculator, begin by clicking the tab appropriate to the calculation you need to perform, either energy stored in a Capacitor or in an Inductor. Then, enter any two known values from the following:

Capacitance (C, Farads) or Inductance (L, Henrys)
Voltage (E, Volts) or Current (I, Amperes)
Energy (W, Joules)
Then click the Calculate button and the other value will be calculated and shown in the appropriate box. Click Clear at any time to clear any current values and start over. Note that the Clear button only effects the tab you are currently on, i.e., if you are on the Capacitor tab and click Clear, any values entered on the Inductor tab will remain.


Impedance
This tool is used to calculate impedance in AC circuits.

To use this calculator, begin by selecting whether you are working with a Serial or Parallel combination of components by clicking the appropriate radio button. Then, enter any two values from the following:

Resistance (R, Ohms)
Reactance (X, Ohms)
Impedance (Z, Ohms)
Then click the Calculate button and the other value will be calculated and shown in the appropriate box. Click Clear at any time to clear any current values and start over.


Morse Code Encoder
This tool will convert a text string to morse code.

To use this enter a string of text into the edit box. You can use the letters A-Z, the numbers 0-9, space (to separate words) as well as the following special characters:

" ' : , - ( ) . ?

Then click the Encode button and you will hear the morse code that represents what you entered. Clicks Clear to clear the edit box and start again.

NOTE: If you do not hear the morse code after clicking Encode, go into your device's control panel (Click the Start Menu, then Settings) and select the Sounds & Reminders applet. Be sure the option Programs is checked, otherwise the morse code will not be heard.


R/C/I Calculator
This tool is used to calculate total resistance, capcitance or inductance in either serial or parallel combinations.

To use this calculator, begin by clicking the tab appropriate to the calculation you need to perform, either Resistance, Capacitance or Inductance. Then, select whether you want the values to be calculated in Serial or Parallel by clicking the appropriate radio button.

Next, enter as many values as needed into the list. To do this, enter a value in the New Value box and click the add (+) button. This will add the new value to the list of Component Values. You can delete a value from the list by clicking it and then clicking the remove (-) button. You can clear all values by clicking the clear (X) button.

When you have finished entering all the values you need, click the Calculate button and the total will be shown in the results box at the bottom.


Reactance
This tool is used to calculate inductive or capacitive reactance in AC circuits.

To use this calculator, begin by clicking the tab appropriate to the calculation you need to perform, either Inductive or Capacitive reactance. Then, enter any two known values from the following:

Frequency (F, Hertz)
Inductance (L, Henrys) or Capacitance (C, Farads)
Reactance (X, Ohms)
Then click the Calculate button and the other value will be calculated and shown in the appropriate box. Click Clear at any time to clear any current values and start over. Note that the Clear button only effects the tab you are currently on, i.e., if you are on the Inductive tab and click Clear, any values entered on the Capacitive tab will remain.


Resistor Color Cpde
This tool is used to determine the value of a resistor based on it's color code.

Begin by setting the colors of the bands of the resistor. This can be done in two ways: you can click the bands on the resistor to cycle through the colors, or select them from the drop-downs. Clicking the bands will update the selection in the appropriate drop-down. When the colors are set properly, click the Calculate button and the resistor's value and low and high tolerance values will be shown below.


Resonance
This tool is used to calculate the resonant frequency of an inductive/capacitive combination.

To use this calculator, simply enter any two known values from the following:

Inductance (L, Henrys)
Capacitance (C, Farads)
Frequency (F, Hertz)
Then click the Calculate button and the other value will be calculated and shown in the appropriate boxes. Click Clear at any time to clear any current values and start over.


Three Phase Connections
This tool is used to calculate values for connecting three-phase power in delta and wye configurations.

To use this calculator, begin by selecting the From and To configurations. Pick either Delta or Wye from both drop-downs. The diagrams below will change accordingly. Next, enter any two known values from the following:

Primary Voltage (Ep, Volts)
Secondary Voltage (Es, Volts)
Turns Ratio (N)
Then click the Calculate button and the other value will be calculated and shown in the appropriate box. Click Clear at any time to clear any current values and start over.


7400 Series Logic Chips
This reference contains data for many of the most common 7400 series logic chips. To use it, make sure you are on the Part Selection tab, then select the part you are interested in from the list at top. A description of the component will be shown below when you click on it.

Click the Pinout tab to see the pinout of the chip.

Click the Function Table tab to see the function table for the function the part performs.


Basic Op-Amp Circuits
This reference contains diagrams of the basic om-amp circuit configurations and important equations relating to each. To use it, simply select the name of the configuraiton from the drop-down box and the circuit will be show below.


Citizens Band Frequencies
This reference contains data for the citizen's band frequencies for ham operators. It shows the channel #, frequency in MHz and the lower and upper tolerances for the band. You can sort the list by clicking the header for any column you wish.


Common Connectors
This reference contains data for many of the most common connectors found in computers and electronics. To use it, simply select the name of the connector from the drop-down. A diagram of the connector will appear below.


Copper Wire Characteristics
This reference contains data for copper wire guages. It shows the AWG #, Nominal Bare Diameter, Nominal Circular Mils, Nominal Bare Feet Per Pound, Nominal Ohms Per 1000 Feet At 20 Degrees Celsius, Current Carrying Capacity At 700 Centimeters Per Amp, Turns Per Linkear Inch Square Film Coat and Turns Per Linear Inch H Film Coat. You can sort the list by clicking the header for any column you wish.


Dialectric Constants
This reference shows the dialetric constant range for many common materials. It shows the Material Name and the Upper and Lower ends of the dialectric range. You can sort the list by clicking the header for any column you wish.


K Factors For Attenuator Loss
This reference shows decibel/K Factor correspondance for use in calculating attenuator loss. It shows the Decibel and K Factor. You can sort the list by clicking the header for any column you wish.


Metal Alloy Resistances
This reference shows the resistance of many common materials. It shows the Material Name, Chemical Symbol and the Resistance. You can sort the list by clicking the header for any column you wish.


Schematic Symbols
This reference shows many of the most common schematic symbols encountered in electronics. Simply scroll through the list to view the symbols and a description of each.


TV Channels
This reference shows data for the most common television channels. It shows the Channel #, Lower Frequency Tolerance, Upper Frequency Tolerance, Video Carrier Frequency, Color Carrier Frequency and Sound Carrier Frequency. You can sort the list by clicking the header for any column you wish.


Radio Frequency Spctrum
This reference shows the Radio Frequency Spectrum of 3 kHz to 3,000,000 MHz. This spectrum is divided into the various bands shown on this screen. sort the list by clicking the header for any column you wish.
